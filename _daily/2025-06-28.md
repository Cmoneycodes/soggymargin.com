# I Am Back

## Hotkeys in Foam / VS Code

### Creating Notes
- **Cmd + Click** (or F12): Create a new note from a [[wikilink]].
- **Cmd + Shift + P** (Ctrl + Shift + P for Windows): Execute "Foam: Create Note" to create a new note with a Title Case Name.
- **Alt + D** (or remap to `editor.action.revealDefinition`): Open your daily note.

### Editing Notes
- **Alt + C**: Change the state of a line to a TODO item.
- **Cmd + B**: Make the selected text bold.
- **Cmd + I**: Make the selected text italic.
- **Alt + Shift + F**: Format a table (place cursor in the table).

### Other Useful Shortcuts
- Ctrl + N: Create a new blank page.
- Ctrl + O: Open a previously saved note.
- Ctrl + S: Save the current note.
- Ctrl + Shift + S: Save all open documents.
- Ctrl + W: Close a document.
- Ctrl + P: Open Print dialog.
- Ctrl + Z: Undo.
- Ctrl + Y or Ctrl + Shift + Z: Redo.
- Ctrl + X / Shift + Del: Cut.
- Ctrl + C / Ctrl + Ins: Copy.
- Ctrl + V / Shift + Ins: Paste.
- Del: Delete selected text.
- Ctrl + A: Select All.
- F5 or Fn + F5: Insert the date and time.
- Ctrl + F: Open Find.
- Ctrl + H: Open Replace.
- Alt + Spacebar, S, then arrow keys: Move or resize the Notepad window.
